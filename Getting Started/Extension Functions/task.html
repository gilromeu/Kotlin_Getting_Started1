Aufgabe:

Erstelle eine Extension Funktion namens invertCase() für die Klasse String, welche einen String zurück gibt, bei dem alle vorherigen Großbuchstaben klein geschrieben und alle Kleinbuchstaben groß geschrieben sind.

Hier könnten folgende Funktionen der Klasse String und CharArray hilfreich sein:
  • isUpperCase()
  • toLowerCase()
  • toUpperCase()
  • map(...)
  • joinToString(...)
