Aufgabe:

  1. Erstelle eine Data Class namens Address, welche (in der richtigen Reihenfolge) die Parameter als unveränderliche Variablen:
    • street vom Typ String
    • houseNumber vom Typ String
    • postalCode vom Typ Int
    • city vom Typ String
    im Konstruktor entgegen nimmt.
  2. Stelle sicher, dass setter & getter entsprechend der Veränderlichkeit zur Verfügung gestellt werden.
  3. Erstelle eine Klasse AddressBook, welche eine veränderliche Liste von Adressen in einer unveränderlichen Variable im Konstruktor entgegen nimmt und passende setter & getter zur Verfügung stellt
    4. Überlege ob AddressBook ein Kandidat für eine Data Class ist.
  4. Vervollständige die Funktion contains(address: Address) der Klasse AddressBook so, dass sie ein Boolean zurück gibt, welches angibt, ob das AddressBook die eingegebene Adresse oder eine Addresse mit den gleichen Werten enthält.
